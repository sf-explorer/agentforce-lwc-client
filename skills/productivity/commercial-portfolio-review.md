# Commercial Portfolio Review

## Prompt

Objective: Visualize insurance contracts by product type and premium value.

Instructions:

1. Use the tableau agent.
2. Query Dataverse Insurance BOB DMO.
3. Display contracts grouped by product type with associated premium totals and counts.
4. Include overall totals and top-performing product categories.

Output format:

- Table: product type, contract count, total premium
- Highlights: top categories by premium

## Expected Result

A portfolio view is produced showing contracts by product type with premium aggregation and top category highlights.
