# Manager Tool: Team Member Monthly Review

## Prompt
Objective: Generate a targeted monthly performance review for one Relationship Manager.

Instructions:
1) List all users with role "Relationship Manager".
2) Ask the manager to select one user.
3) Analyze that user's activity over the previous month.
4) Report key KPIs and major points of attention.

Output format:
- Selected Relationship Manager
- Monthly KPI dashboard
- Key strengths
- Points of attention and suggested actions

## Expected Result
The manager selects a Relationship Manager and receives a structured monthly review with KPIs, strengths, and attention points.
