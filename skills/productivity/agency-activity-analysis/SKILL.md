# Agency Activity Analysis

## Instructions

Objective: Review team member performance for the last month.

Instructions:
1) List all salespeople available for analysis.
2) Ask the user to choose one profile.
3) Analyze that profile's activity over the last 30 days.
4) Report key KPIs (volume, conversion, signed value, pipeline health).
5) Highlight top risk indicators and priority coaching actions.

Output format:
- Selected profile
- KPI summary
- Top 3 risk indicators
- Recommended manager actions
