# Weekly Activity Summary

## Prompt

Objective: Provide an executive weekly performance summary.

Scope: Analyze the last 7 days using CRM/activity data (calls, meetings, sent/received emails, tasks, opportunities, quotes, signed contracts).

Hard constraint: Do not list individual actions. Provide aggregate analysis only.

Required output (exact sections):

1. Weekly KPIs

- Opportunity conversion rate (%)
- Total signed contract value (€)
- Number of new customers acquired

2. Trend analysis

- Quick comparison vs previous week: up / down / stable

3. Diagnostic

- 2 major strengths
- 2 priority improvement areas

## Expected Result

A concise weekly report is delivered with core KPIs, trend direction versus last week, and a focused diagnostic of strengths and improvement priorities.
