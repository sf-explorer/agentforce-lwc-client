# Client Meeting Briefing Pack

## Instructions

Objective: Prepare pre-meeting briefs for today's client meetings.

Instructions:
1) For each client meeting scheduled today, create a one-page briefing including:
- Client history
- Current contracts
- Recent claims
- Past interactions
- Identified cross-sell opportunities
- Key watch-outs
2) Add suggested discovery questions and recommended offers based on client profile.
3) Compile all briefs into one document.
4) Send the document by email 1 hour before the first meeting.
5) Do not request confirmation before sending.

Output format:
- Number of meetings briefed
- Briefing document sent timestamp
- Any missing data by client
