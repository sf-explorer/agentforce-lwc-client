# Renewal Follow-up

## Instructions

Objective: Increase on-time contract renewals.

Scope: Find contracts expiring in the next 30 days that are not yet renewed.

Instructions:
1) Retrieve all eligible contracts.
2) For each customer, draft a personalized reminder email with: expiry date, current coverage highlights, and a proposal to schedule a renewal meeting.
3) Send emails immediately via CRM Assistant (no approval required).
4) Create a follow-up task for 7 days later if no response is received.
5) Notify the manager with the number of emails sent.

Output format:
- Contracts identified
- Emails sent
- Follow-up tasks scheduled
- Exceptions/errors
