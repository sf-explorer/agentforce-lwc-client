# Multi-Agent Orchestration

## Instructions

Objective: Build a quick consolidated report for customer Annie Dubois using multiple agents.

Instructions:
1) Use the GED agent to list Annie Dubois' identity documents.
2) Use the Solaris agent to retrieve Annie Dubois' claims.
3) Produce a concise consolidated report with:
- Identity documents found
- Claims summary
- Key risks or missing information
- Recommended next action

Output format:
- GED results
- Solaris results
- Consolidated summary
